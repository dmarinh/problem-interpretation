# Appendix: Source References for Sensitivity Analysis Queries

This appendix provides download links, exact locations (where determinable), and access notes for every source document referenced in the sensitivity analysis query design document. It should be read alongside the main document `sensitivity_analysis_queries.md`.

**Precision notation used throughout:**
- **EXACT**: The page number, section, or table cited has been verified against the actual document.
- **APPROXIMATE**: The document is confirmed correct, but the exact page/section is estimated based on table of contents or secondary references, not direct access to the passage.
- **DOCUMENT-LEVEL**: The document is the correct source, but I cannot pinpoint the exact location of the relevant content within it.
- **INFERRED**: The source basis is derived from well-established domain knowledge and standard regulatory practice. No single source document was used; rather, the scenario type is canonical in the field.
